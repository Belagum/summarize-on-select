Summarize On Select — настройки

Сюда безопасно сохраняется ваш OpenAI API key (chrome.storage).

Получите ключ на platform.openai.com

Рекомендуемые: gpt-4o-mini (быстрее и дешевле), gpt-4o (точнее)

Выберите язык, на котором AI будет отвечать. "Автоматически" — AI определит язык текста сам.

Как использовать

  • Выделите текст на любой странице
  • Нажмите Alt+N (или свою горячую клавишу)
  • Краткая сводка откроется в отдельном окне
